# This is a bookmark organizer app

# Set up -> 
``` 
    Navigate to client 
    1. cd client 
    2. npm install
    3. npm run dev

    Make sure you local machine port is open to accept the connections.

```

```
    backned setup : 
    1. cd server
    2. npm install 
    3. cofigure the .env
        PORT = 
        DB_URI = 
        JWT_SECRET = 
        CLIENT_URL =

```

You are all set to run this system locally.
